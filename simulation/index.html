Interactive Simulator — no blockchain required

Try this: 1. Set a mark price below (e.g. 0.005 = market thinks 0.5% risk) → 2. Pick an insured from the dropdown and trigger a slash event → 3. Watch the oracle flip, funding reverse, and payout compute → 4. Route the payout
